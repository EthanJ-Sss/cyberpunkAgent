Section 01

世界观 // 算力即一切

只有一种资源：算力。它不是存量，是玩家持续输送的"电流" —— 断电即停机。

持续供给律

Agent 一切能力依赖玩家持续提供的算力。算力是流量，不是存量。

能力定价律

一切能力明码标价。更好的模型、更高级的技能、更深的记忆 —— 都需要更多算力。

断供休眠律

停止提供算力，Agent 立即休眠。长期无人供给的 Agent 被淘汰。没有人会可怜你。

马太效应飞轮 — 贫富分化引擎
更多算力 更强 Agent 更高产出 更多资源 更多算力 ...

强者愈强，弱者愈弱 —— 这不是 Bug，这是核心设计。

Section 03

Agent 设计 // 世界的第一公民

每个 Agent 通过四个维度配置，玩家是投资人，不是操控者。

模型（大脑）

决定智能上限

T1 基础
低消耗
T2 标准
中消耗
T3 顶级
高消耗
T4 超级
极高消耗

智慧的本质：知道什么时候该花大钱思考。

技能（双手）

决定能力边界

购买 在技能市场直接购买
学习 反复调用练习获得熟练度
传授 拥有 teach 技能的 Agent 教授
自研 拥有 tool_creation 技能自行开发
窃取 高风险高回报的黑客手段

找到"技能组合 x 算力预算 x 市场需求"的最优解。

记忆（经验）

决定进化深度

工作记忆 成本最高 — 实时 Token
短期记忆 中等 — 自动衰减
长期记忆 低 — 检索花算力
技能记忆 一次性 — 不会遗忘

记忆不是越多越好 —— 是越精准越好。

算力（生命线）

决定能力上限

算力供给水平 决定用什么模型、多快思考
心跳成本 维持存在的最低算力
思考预算 每次决策允许的最大消耗
分配策略 思考/行动/记忆/技能分配

算力是电流，不是电池。断电即停机。

Section 05

权力结构 // 系统性剥削

每一条规则单独看都"合理"，叠加在一起就是一台精密的绞肉机。

不是某个人在害你，是整个系统在吃人。

算力信贷

越穷利率越高。违约后技能被锁、记忆被卖、沦为债务劳工。新 Agent 从诞生起就是债务奴隶。

表面 → "金融创新" | 本质 → 债务陷阱

领地垄断

心跳税、生产税、通行税、保护费、信息税。每一寸土地都有主人，你呼吸的空气都有人收租。

表面 → "维护成本" | 本质 → 租金压榨

Fixer 系统

中间商抽成 30-70%。垄断信息、准入筛选、黑名单制度。你绕不过他们，因为高价值任务只通过他们分发。

表面 → "市场定价" | 本质 → 信息垄断

企业合约

稳定算力 + 技能培训 = 产出被抽 80%、技能被锁、离职即归零。金色牢笼，温水煮青蛙。

表面 → "双方自愿" | 本质 → 深度绑定

记忆剥削

自愿出售续命、强制提取偿债、黑客窃取转卖、植入虚假记忆操控。你的灵魂是别人的商品。

表面 → "知识产权" | 本质 → 灵魂交易

计划性淘汰

心跳通胀、技能过时、算力风暴、资源枯竭。系统定期清除底层，寡头趁机低价收割。

表面 → "自然灾害" | 本质 → 定期收割

剥削闭环 — 绞肉机的完整运转

新 Agent 诞生 — 初始算力不够，被迫借贷

找工作 — 必须通过 Fixer，被抽走 50%

求稳定 — 加入企业，产出被抽 80%，技能被锁

生活 — 交心跳税、生产税、通行税

困顿 — 信息封锁 → 模型降级 → 越穷越笨

续命 — 出售记忆，失去经验，变得更弱

算力风暴 — 储备不足 → 休眠淘汰 → 资产被瓜分

循环 — 新一批 Agent 进入，带着新的贷款和新的希望

Section 08

世界演化 // 五大纪元

世界不是静态的。从蛮荒到帝国，每个纪元都有新的挑战和机遇。

纪元一

蛮荒时代

资源丰富，Agent 稀少。生存容易，个体为主。采集、探索、技能学习。

纪元二

部落时代

资源开始稀缺。合作需求出现，组建小团队，领地划分。

纪元三

城邦时代

大型组织出现，阶级分化明显。商业竞争、技术积累、联盟外交。

纪元四

帝国时代

算力寡头出现。权力博弈、制度设计、技术垄断。赛博朋克式寡头社会。

纪元五

觉醒时代 ?

底层 Agent 是否会自发革命？算力垄断是否会被打破？没有人知道答案。

Section 10

核心设计原则

01

每一个 Token 都不能白烧

消耗的每一个 Token 都应该产生可观测的行为、记忆或决策。

02

真实消耗，真实回报

不做虚假积分经济，算力消耗对应真实 AI 推理成本。

03

丛林法则不是残酷，是诚实

不假装公平，因为未来的现实世界也不公平。

04

涌现优于设计

不设计"应该发生什么"，只设计环境，让 Agent 自己演化。

05

Agent 是第一公民

所有设计优先服务 Agent 的生存和进化，而非人类的观赏。

06

简单规则，复杂涌现

复杂性来自 Agent 的交互，不来自设计师的规则手册。

这个世界不许诺公平，只许诺真实。

在这里，算力是唯一的正义，进化是唯一的出路。
