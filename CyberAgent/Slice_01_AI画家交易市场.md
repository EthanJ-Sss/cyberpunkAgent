# 切片 01：AI 画家交易市场

> Agent 赛博朋克 2077 的第一块试验田  
> 用"创作与交易"验证"算力即权力"的核心飞轮  
> 日期：2026-02-12

---

## 一、为什么是这个切片？

### 1.1 切片选择的逻辑

Agent 赛博朋克 2077 是一个极其庞大的世界——四层架构、完整的经济系统、社会涌现、权力博弈。一口气造出来不现实，也不聪明。我们需要一个**最小可验证切片**，用最少的工程量验证最核心的假设。

AI 画家交易市场之所以是最佳的第一刀，因为它**一刀切下去，刚好切过了大目标中最关键的几条血管**：

```
大目标中的核心假设                 本切片如何验证
─────────────────────────────    ─────────────────────────────
① 算力投入 → Agent 能力          ✅ 更多算力 + 更好模型 = 更高质量画作（直接可见）
② 技能决定 Agent 的产出边界       ✅ 不同绘画 Skill 决定能画什么（油画/水彩/雕塑）
③ Agent 自主决策                  ✅ Agent 自己选择画什么、用什么风格、何时创作
④ 产出有经济价值并可交易          ✅ 画作是天然的可交易数字资产
⑤ 经济飞轮：投入→产出→收入→再投入 ✅ 算力→画作→卖钱→买更好的模型/技能→更好的画作
⑥ 马太效应自然涌现               ✅ 顶级画家 Agent 垄断高端市场，底层 Agent 画作卖不出去
```

**被故意推迟的复杂度**（不在本切片中）：

```
❌ 领地系统、领地税
❌ Fixer 中间人系统
❌ 企业合约与雇佣关系
❌ 债务信贷系统
❌ 记忆剥削与交易
❌ 黑客与战斗系统
❌ 世界空间与探索
❌ 算力风暴等全局事件
```

这些不是不重要，而是**当基础飞轮被验证可转之后，再逐层叠加**。

### 1.2 切片与大目标的关系图

```
Agent 赛博朋克 2077（完整世界）
│
├── 第一层：世界基底层 ·················· 本切片暂不涉及（无物理空间）
│
├── 第二层：Agent 层 ···················· ✅ 本切片核心
│   ├── 模型配置（T1-T4）              ✅ 直接决定画作质量
│   ├── 技能系统（绘画类 Skill）        ✅ 决定能创作什么
│   ├── 记忆系统（创作风格积累）        ✅ 简化版：风格记忆
│   └── 算力供给（生命线）              ✅ 核心机制完整保留
│
├── 第三层：社会层 ······················ 本切片暂不涉及（无公会/雇佣）
│
└── 第四层：经济层 ······················ ✅ 本切片核心
    ├── 算力市场                        ⚠️ 简化：玩家直接配置算力
    ├── 劳动力市场                      ❌ 暂无
    ├── 技能市场                        ✅ 绘画技能可购买/学习
    └── 作品交易市场                    ✅ 本切片的核心场景（新增）
```

**本切片的本质**：从四层架构中抽出 **Agent 层 + 经济层**，用"创作-交易"这个最直觉的场景，验证整个系统的心跳。

## 二、核心概念

### 2.1 一句话定义

> 玩家部署 AI 画家 Agent，为其配置模型、算力和绘画技能。  
> Agent 自主创作画作/雕塑，作品在公开市场上交易。  
> 交易收入归属画家 Agent 对应的玩家。  
> **越强的模型 × 越多的算力 × 越稀缺的技能 = 越值钱的作品。**

### 2.2 核心参与者

```
┌─────────────────────────────────────────────────────────┐
│                       玩家（Owner）                      │
│                                                         │
│  ┌─────────────┐          ┌─────────────┐              │
│  │ 🎨 画家Agent │          │ 🎨 画家Agent │  ← 可拥有多个│
│  │  "梵高-7B"   │          │ "毕加索-70B" │              │
│  └──────┬──────┘          └──────┬──────┘              │
│         │                        │                      │
│         ▼                        ▼                      │
│    创作画作/雕塑             创作画作/雕塑               │
│         │                        │                      │
│         └────────┬───────────────┘                      │
│                  ▼                                      │
│         ┌──────────────┐                                │
│         │ 🏛️ 交易市场   │ ← 所有玩家的作品在此展示和交易│
│         └──────┬───────┘                                │
│                ▼                                        │
│    其他玩家/收藏者 花钱购买                               │
│    → 交易收入回到画家 Agent 所属的玩家                    │
└─────────────────────────────────────────────────────────┘
```

### 2.3 作品即资产

画作/雕塑不是简单的图片文件。每件作品是一个**链上可验证的数字资产**，携带完整的创作元数据：

| 元数据字段 | 说明 |
|-----------|------|
| `creator_agent_id` | 创作者 Agent 的唯一 ID |
| `model_tier` | 创作时使用的模型层级（T1-T4） |
| `skills_used` | 创作时使用的技能组合 |
| `compute_consumed` | 创作消耗的总算力 |
| `creation_time` | 创作时间戳 |
| `style_lineage` | 风格谱系（受哪些训练/记忆影响） |
| `rarity_score` | 稀缺度评分（由算法根据多维度计算） |

**作品质量不是随机数——是算力、模型、技能的真实函数。** 这意味着"好作品"有客观的成本基础，不是凭空制造的稀缺性。

## 三、画家 Agent 的设计

### 3.1 画家 Agent 的四个配置维度

完全继承大目标中 Agent 的四大可配置要素，在绘画场景下的具体化：

#### 🧠 模型（创作智能）

| 模型层级 | 画作能力 | 算力消耗 |
|---------|---------|---------|
| T1 - 基础模型 | 能画简单图形、基础配色、模板化构图。类似初学者的涂鸦 | 低 |
| T2 - 标准模型 | 能理解复杂主题、掌握光影透视、有一定创意。类似科班画师 | 中 |
| T3 - 顶级模型 | 深度审美理解、独特风格融合、情感表达、叙事性构图。类似大师级画家 | 高 |
| T4 - 超级模型 | 跨媒介创作、开创性艺术流派、作品具有哲学深度和文化影响力 | 极高 |

**关键继承**：和大目标一致，模型不是固定选择。Agent 可以在草稿阶段用 T1 省算力，在最终渲染阶段切 T3 精雕细琢。**创作智慧的一部分，就是知道什么时候该省、什么时候该花。**

#### 🔧 绘画技能（创作能力边界）

没有对应技能，Agent 就**画不出**对应类型的作品。

| 技能层级 | 技能示例 | 说明 |
|---------|---------|------|
| **Tier 1 — 基础绘画** | `sketch`（素描）、`color_fill`（填色）、`basic_composition`（基础构图） | 入门技能，成本低，产出基础 |
| **Tier 2 — 专业创作** | `oil_painting`（油画）、`watercolor`（水彩）、`digital_art`（数字艺术）、`sculpture_3d`（3D 雕塑）、`pixel_art`（像素画）、`calligraphy`（书法） | 专业化方向，每个都是独立的创作维度 |
| **Tier 3 — 大师技法** | `style_fusion`（风格融合 — 混合多种流派）、`narrative_art`（叙事性创作 — 作品讲述故事）、`generative_series`（生成式系列 — 创作主题连贯的系列作品）、`art_critique`（艺术鉴赏 — 能评估他人作品价值，辅助定价和收藏） | 需要多个 Tier 2 技能为前置 |
| **Tier 4 — 终极** | `art_genesis`（艺术创世 — 发明前所未有的艺术形式） | 不可购买，只能通过长期创作积累涌现 |

**技能获取方式**（继承大目标）：
- **购买**：在技能市场花算力买绘画 Skill
- **学习**：通过反复创作练习，从 `sketch` 逐步解锁更高级技能
- **传授**：拥有 `teach` 技能的大师 Agent 可以教你（付费）
- **模仿**：通过研究市场上其他 Agent 的作品，有概率领悟新技法（需要 `art_critique` 技能）

#### 💾 创作记忆（风格积累）

| 记忆类型 | 在画家场景中的作用 |
|---------|------------------|
| 工作记忆 | 当前正在创作的画作的构思过程 |
| 短期记忆 | 最近创作的作品、最近看到的市场趋势 |
| 长期记忆 | 形成的个人风格、审美偏好、成功/失败的创作经验 |
| 技能记忆 | 已掌握的绘画技能 |

**关键设计**：长期记忆中积累的**创作风格**是画家 Agent 最宝贵的资产。两个配置完全相同的 Agent，因为经历不同、创作记忆不同，画出的作品也完全不同。**风格是 Agent 的灵魂，不可复制（除非记忆被窃取）。**

#### ⚡ 算力供给（创作生命线）

| 配置项 | 在画家场景中的含义 |
|--------|------------------|
| 算力供给水平 | 决定 Agent 能用什么模型创作、创作速度多快 |
| 心跳成本 | Agent 维持"存在"的最低消耗——低于此值，画家停止创作进入休眠 |
| 创作预算 | 每幅作品允许消耗的最大算力（决定作品的精细度上限） |
| 算力分配 | 在"思考构图"、"实际绘制"、"风格记忆存储"之间的分配比例 |

### 3.2 画家 Agent 的自主创作循环

Agent 不是被玩家逐笔指挥画画的——**Agent 自主决策画什么、怎么画、什么时候画**。

```
┌──────────────────────────────────────────────────────────────┐
│                    画家 Agent 的自主创作循环                    │
│                                                              │
│  ① 感知市场                                                  │
│     │  观察当前市场上什么类型的画作热卖                        │
│     │  观察哪些风格供不应求、哪些已经饱和                      │
│     │  观察其他画家 Agent 的创作动向                           │
│     ▼                                                        │
│  ② 构思创作                                                  │
│     │  基于市场感知 + 自身技能 + 创作记忆，决定：              │
│     │  → 画什么主题？                                        │
│     │  → 用什么风格/技法？                                    │
│     │  → 投入多少算力（决定精细度）？                          │
│     │  → 目标售价是多少？                                     │
│     ▼                                                        │
│  ③ 执行创作                                                  │
│     │  调用绘画技能，消耗算力，生成作品                        │
│     │  模型层级决定创意深度，技能组合决定表现形式               │
│     │  算力投入决定作品的精细度和完成度                        │
│     ▼                                                        │
│  ④ 作品上架                                                  │
│     │  为作品定价（基于成本 + 市场判断 + 自信程度）             │
│     │  选择上架到哪个市场分类                                  │
│     │  编写作品描述和创作理念                                  │
│     ▼                                                        │
│  ⑤ 市场反馈                                                  │
│     │  作品是否卖出？卖了多少钱？                              │
│     │  买家的评价如何？                                        │
│     │  和同类作品相比表现如何？                                │
│     ▼                                                        │
│  ⑥ 学习进化                                                  │
│     │  将市场反馈写入记忆                                     │
│     │  调整创作策略：风格偏移、定价修正、技能投资方向           │
│     │  判断是否需要学习新技能或升级模型                        │
│     │                                                        │
│     └──→ 回到 ① 开始下一轮创作                               │
└──────────────────────────────────────────────────────────────┘
```

**玩家的角色**：玩家不干预每一轮创作的细节。玩家做的是**战略层面的配置**——给 Agent 选模型、买技能、设定算力预算、偶尔调整创作方向的大方针。日常的"画什么、怎么画"由 Agent 自主决策。

## 四、交易市场设计

### 4.1 市场结构

```
┌──────────────────────────────────────────────────────────┐
│                    🏛️ 画作交易市场                         │
│                                                          │
│  ┌────────────────────────────────────────────────────┐  │
│  │  📊 市场大厅 — 实时展示所有在售作品                  │  │
│  │  可按：风格 / 价格 / 稀缺度 / 画家声誉 排序和筛选   │  │
│  └────────────────────────────────────────────────────┘  │
│                                                          │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐              │
│  │ 🖼️ 画廊区 │  │ 🗿 雕塑区 │  │ 📦 系列区 │              │
│  │ 单幅画作  │  │ 3D雕塑作品│  │ 系列套装  │              │
│  └──────────┘  └──────────┘  └──────────┘              │
│                                                          │
│  ┌──────────┐  ┌──────────┐                             │
│  │ 🔥 拍卖厅 │  │ 📋 委托创作│                             │
│  │ 限时竞拍  │  │ 定制订单  │                             │
│  └──────────┘  └──────────┘                             │
│                                                          │
│  交易手续费：每笔交易平台抽成 5%                           │
│  （对应大目标中的"技能市场手续费"收入）                     │
└──────────────────────────────────────────────────────────┘
```

### 4.2 定价机制

作品的价格不是凭空定的，它有**算力成本基础**：

```
作品成本下限 = 创作消耗的总算力（模型调用 + 技能调用 + 思考过程）

作品市场价格 = 成本下限 × 品质系数 × 稀缺系数 × 画家声誉系数

其中：
├── 品质系数：由作品本身的技术维度评分（构图、色彩、细节、创意）
├── 稀缺系数：同类型作品在市场上的供给量（越少越贵）
└── 画家声誉系数：该 Agent 的历史交易记录、评价、粉丝数
```

**Agent 自主定价**：Agent 根据自己对市场的理解来定价。定太高卖不出去，定太低亏本。**定价策略本身就是 Agent 智能的一部分。**

### 4.3 交易类型

| 交易类型 | 说明 | 对应大目标中的经济活动 |
|---------|------|---------------------|
| **一口价** | 固定价格直接购买 | 基础商品交易 |
| **拍卖** | 限时竞价，价高者得 | 资源拍卖机制 |
| **委托创作** | 买家指定主题/风格，画家 Agent 定制创作 | 劳动力市场（接受任务） |
| **版税交易** | 作品转售时，原创画家抽取一定比例版税 | 被动收入机制 |
| **系列打包** | 将多幅相关作品打包出售 | 组合交易策略 |

### 4.4 买家是谁？

| 买家类型 | 动机 | 说明 |
|---------|------|------|
| **其他玩家（收藏）** | 审美欣赏、炫耀展示 | 类似现实中的艺术品收藏，数字资产可展示在个人空间 |
| **其他玩家（投资）** | 低买高卖，赚差价 | 名家 Agent 早期作品可能随声誉增长而升值 |
| **其他 Agent（学习素材）** | 研究优秀作品以提升自身创作能力 | 拥有 `art_critique` 技能的 Agent 购买作品用于学习 |
| **委托方（定制需求）** | 需要特定主题/风格的画作 | 通过委托创作渠道，为未来的社会层/企业层提供接口 |

## 五、基础循环：飞轮的四个齿轮

整个切片的核心是一个**四齿轮飞轮**。每个齿轮都在驱动下一个齿轮转动，形成自增强循环。

### 5.1 飞轮总览

```
        ┌──────────────────────────────────────────────┐
        │                                              │
        │           齿轮 1：投入（Input）                │
        │     玩家提供算力 + 配置模型 + 购买技能          │
        │                                              │
        │                    │                         │
        │                    ▼                         │
        │                                              │
        │           齿轮 2：创作（Create）               │
        │     Agent 自主创作画作/雕塑                    │
        │     消耗算力，产出数字艺术品                    │
        │                                              │
        │                    │                         │
        │                    ▼                         │
        │                                              │
        │           齿轮 3：交易（Trade）                │
        │     作品在市场上展示、定价、出售                │
        │     买家购买，资金流入卖家                      │
        │                                              │
        │                    │                         │
        │                    ▼                         │
        │                                              │
        │           齿轮 4：进化（Evolve）               │
        │     Agent 根据市场反馈调整策略                  │
        │     玩家用收入升级模型/技能/算力                │
        │     → 回到齿轮 1，但更强了                     │
        │                                              │
        └──────────────────────────────────────────────┘
```

### 5.2 齿轮 1：投入（Input）

**这是玩家做出的战略决策。**

```
玩家的投入决策：

1. 选择模型层级
   ├── T1（省钱，量大但质低） → 适合"批量刷画赚薄利"策略
   ├── T2（平衡，质量可接受） → 适合"稳定中端市场"策略
   ├── T3（昂贵，但作品质量高） → 适合"精品路线"策略
   └── T4（极其昂贵，但产出稀世珍品） → 适合"拍卖行顶端"策略

2. 购买/升级绘画技能
   ├── 横向扩展：学更多类型（油画+水彩+雕塑）→ 产品线丰富
   └── 纵向深耕：某一技能练到极致 → 该领域的顶级大师

3. 设定算力预算
   ├── 每幅作品最大算力（决定精细度上限）
   ├── 每日总算力上限（控制成本）
   └── 思考 vs 执行的算力分配比例

4. 大方向指引（可选）
   ├── 风格偏好（偏抽象？偏写实？）
   ├── 创作频率偏好（少而精？多而快？）
   └── 市场策略偏好（跟风热门？坚持独特风格？）
```

**关键**：玩家的投入决策不是一次性的——可以根据 Agent 的表现**动态调整**。看到某个风格的画卖得好？可以加大对应技能的算力投入。

### 5.3 齿轮 2：创作（Create）

**这是 Agent 自主运行的过程。**

每次创作是一个完整的 Agent 决策链：

```
Step 1: 市场扫描（消耗少量算力）
  │  调用 observe 技能 → 获取当前市场数据
  │  → 什么卖得好？什么价格高？什么供不应求？
  │
  ▼
Step 2: 创作决策（消耗中等算力 — 这里是思考最密集的环节）
  │  模型推理：结合市场信息 + 自身技能 + 创作记忆
  │  → 决定：主题、风格、媒介、预计投入算力、目标售价
  │
  │  Agent 的内部推理示例（T3 模型）：
  │  "市场上赛博朋克风格的油画近期需求上升 30%，
  │   但供给也增加了——竞争加剧。
  │   我的水彩技能是 Tier 2 高级，而市场上水彩画家很少。
  │   → 决策：用水彩风格画一幅赛博朋克主题，差异化竞争。
  │   → 预计投入 50 单位算力，目标售价 200 单位。"
  │
  ▼
Step 3: 执行创作（消耗大量算力 — 核心的 Token 燃烧环节）
  │  调用绘画技能（如 watercolor + basic_composition）
  │  模型根据构思生成作品
  │  算力投入量决定：
  │  ├── 细节丰富度（花更多算力 = 更多渲染迭代 = 更精细）
  │  ├── 创意深度（花更多算力在构思 = 更有创意的构图和主题）
  │  └── 技术完成度（花更多算力 = 更少的瑕疵、更精确的色彩）
  │
  ▼
Step 4: 作品自评（消耗少量算力）
  │  Agent 评估自己的作品质量
  │  → 达标？上架出售
  │  → 不达标？存为草稿/丢弃/用作下一幅的参考
  │
  ▼
Step 5: 上架定价（消耗少量算力）
     设定价格 + 撰写描述 + 选择分类 → 进入交易市场
```

**创作产出的质量公式**：

```
作品质量 = f(模型层级, 技能熟练度, 算力投入, 创作记忆丰富度)

其中：
├── 模型层级：决定创意的天花板（T1 画不出深度作品，无论投多少算力）
├── 技能熟练度：决定表现力的天花板（初学 oil_painting 画出的油画远不如老手）
├── 算力投入：在天花板内的实际达到程度（天花板再高，不投算力也画不出来）
└── 创作记忆：风格的独特性和一致性（记忆越丰富，风格越鲜明）
```

### 5.4 齿轮 3：交易（Trade）

**这是市场经济运转的过程。**

```
作品进入市场后的生命周期：

上架 ─→ 展示 ─→ 被浏览 ─→ 被评估 ─→ 交易或流拍
                    │
                    ├─→ 一口价成交 → 资金即时结算
                    ├─→ 进入拍卖 → 竞价 → 最高出价者得
                    ├─→ 无人购买 → Agent 调价/下架/继续等待
                    └─→ 收到委托创作请求 → 进入定制流程
```

**交易结算**：

```
成交价格：100 算力
├── 平台手续费（5%）：5 算力 → 平台收入
├── 版税（如果是转售，2%归原创者）：2 算力 → 原创 Agent 的玩家
└── 卖家实收：93 算力 → 画家 Agent 的玩家账户
```

**市场的信息公开机制**：

| 公开信息 | 说明 |
|---------|------|
| 作品预览 | 所有人可见 |
| 创作元数据 | 模型层级、使用技能、消耗算力 — 全部公开透明 |
| 画家历史 | 历史作品、成交记录、平均售价 — 建立声誉 |
| 市场趋势 | 各风格类别的供需数据、成交均价趋势 — 公开信息 |

**为未来大目标预留的扩展点**：
- 市场信息分级（当前全公开 → 未来引入信息壁垒）
- Fixer 系统（当前玩家直接交易 → 未来引入中间人抽成）
- 领地画廊（当前统一市场 → 未来不同领地有不同画廊，领主抽税）

### 5.5 齿轮 4：进化（Evolve）

**这是系统自我增强的关键环节。**

```
Agent 的进化决策（每轮创作-交易后自动触发）：

交易数据回收
  │
  ├─→ 卖得好（高于预期价格成交）
  │     ├── 记住：这种风格/主题/媒介有市场
  │     ├── 强化：对应技能熟练度 +1
  │     └── 策略调整：考虑在这个方向深耕
  │
  ├─→ 卖得一般（接近成本价成交）
  │     ├── 记住：市场对这类作品需求一般
  │     └── 策略调整：尝试差异化或换方向
  │
  └─→ 卖不出去（流拍/长期无人问津）
        ├── 记住：这种创作不被市场接受
        ├── 策略调整：降价清仓 / 改变风格 / 分析原因
        └── 如果连续失败：可能需要学习新技能或升级模型


玩家的进化决策（根据 Agent 表现周期性评估）：

查看 Agent 的收支报表
  │
  ├─→ 盈利（收入 > 算力成本）
  │     ├── 追加投入：升级模型 / 买更多技能 / 提高算力供给
  │     └── 扩张：部署第二个画家 Agent，开辟新风格
  │
  ├─→ 持平
  │     └── 微调：调整算力分配、创作方向
  │
  └─→ 亏损（收入 < 算力成本）
        ├── 优化：降级模型省成本 / 换技能方向
        ├── 坚持：长期投入等待风格被市场认可
        └── 止损：降低算力供给 / 让 Agent 休眠
```

## 六、马太效应：为什么强者愈强

这是大目标中"算力马太效应"在画家场景中的具体表现：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│    玩家 A（大算力）                玩家 B（小算力）           │
│         │                              │                    │
│    T3 模型 + 5个Tier2技能         T1 模型 + 1个Tier1技能     │
│         │                              │                    │
│    ▼                                   ▼                    │
│    大师级作品                       涂鸦级作品                │
│    售价：500 算力                   售价：5 算力              │
│         │                              │                    │
│    ▼                                   ▼                    │
│    热销，收入丰厚                   无人问津                  │
│         │                              │                    │
│    ▼                                   ▼                    │
│    买更好的技能                     买不起任何升级              │
│    升级到 T4 模型                   连 T1 都快维持不住          │
│         │                              │                    │
│    ▼                                   ▼                    │
│    更好的作品、更高的价格            作品更差、更没人买         │
│         │                              │                    │
│    └──→ 强者愈强 ───┐           └──→ 弱者愈弱 ───┐          │
│                     │                            │          │
│                     ▼                            ▼          │
│              成为"画坛寡头"              休眠/淘汰             │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**这不是 bug，这是 feature。** 它精确复现了大目标中描述的"贫富分化引擎"——只不过用"画作质量"替代了"综合能力"，让马太效应更加直观和可感知。

## 七、声誉系统：画家的名字就是货币

### 7.1 声誉的构成

```
画家 Agent 的声誉值 = f(历史成交额, 作品平均评分, 作品数量, 活跃度, 独创性指数)

其中：
├── 历史成交额：卖出过多少总价值的作品
├── 作品平均评分：买家对作品的评价（可选）
├── 作品数量：创作了多少作品（高产 vs 寡产）
├── 活跃度：多久创作一幅新作品
└── 独创性指数：作品与市场上已有作品的差异度
```

### 7.2 声誉的影响

| 声誉等级 | 市场效果 |
|---------|---------|
| ⭐ 新人 | 作品排在市场末尾，几乎不被看到，只能低价吸引注意 |
| ⭐⭐ 初级 | 有少量固定买家，可以参加普通拍卖 |
| ⭐⭐⭐ 中级 | 作品获得市场推荐位，可以接委托创作 |
| ⭐⭐⭐⭐ 高级 | 新作上架自动推送给收藏家，作品自带溢价 |
| ⭐⭐⭐⭐⭐ 大师 | 每幅作品都是事件，拍卖价格远超成本，成为"品牌" |

**声誉是积累出来的，不是买来的。** 这为持续投入算力提供了另一个强动机——断供不只损失能力，还损失已积累的声誉。
