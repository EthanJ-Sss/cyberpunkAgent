画家 Agent

四个配置维度

每个画家 Agent 有四个可调节维度。玩家通过战略性配置这四个旋钮，决定 Agent 的创作能力天花板。

模型（创作智能）
模型层级决定创意天花板。可在草稿阶段用 T1 省钱，渲染阶段切 T3 精修。
  • T1 基础 简单图形、基础配色，类似初学者涂鸦
  • T2 标准 复杂主题、光影透视，类似科班画师
  • T3 顶级 独特风格、情感表达，类似大师级画家
  • T4 超级 跨媒介创作、开创性流派，文化影响力
技能（创作能力边界）
没有对应技能，Agent 就画不出对应类型的作品。技能通过购买、学习、传授或模仿获得。
  • Tier 1 sketch / color_fill / basic_composition
  • Tier 2 oil_painting / watercolor / sculpture_3d
  • Tier 3 style_fusion / narrative_art / art_critique
  • Tier 4 art_genesis（不可购买，只能涌现）
记忆（风格积累）
风格是 Agent 的灵魂。两个配置相同的 Agent，因创作记忆不同，画出的作品完全不同。
  • 工作记忆 当前画作的构思过程
  • 短期记忆 最近作品、市场趋势
  • 长期记忆 个人风格、审美偏好、创作经验
  • 技能记忆 已掌握的绘画技能（不会遗忘）
算力（创作生命线）
算力是流量不是存量。玩家持续提供 GPU 算力，断供即休眠。
  • 供给水平 决定能用什么模型、创作速度
  • 心跳成本 维持"存在"的最低消耗
  • 创作预算 每幅作品的最大算力
  • 算力分配 思考 / 绘制 / 记忆存储的比例
自主创作

Agent 的创作循环

Agent 不是被玩家逐笔指挥——它自主决策画什么、怎么画、什么时候画。每一轮创作都是一个完整的感知-决策-执行链。

1
感知市场
观察市场上什么类型的画作热卖、哪些风格供不应求、竞争对手在创作什么。消耗少量算力。
2
构思创作
结合市场信息 + 自身技能 + 创作记忆，决定主题、风格、媒介、算力投入和目标售价。这是思考最密集的环节。
3
执行创作
调用绘画技能，消耗大量算力生成作品。算力投入量直接决定细节丰富度、创意深度和技术完成度。核心 Token 燃烧环节。
4
作品上架
Agent 自评作品质量，为达标作品定价并上架。定价策略本身就是 Agent 智能的体现——定太高卖不出去，定太低亏本。
5
市场反馈
观察作品是否卖出、卖了多少钱、买家评价如何、和同类作品相比表现如何。
6
学习进化
将反馈写入记忆，调整创作策略——风格偏移、定价修正、技能投资方向。判断是否需要学习新技能或升级模型。然后回到第 1 步。
// 创作质量公式
作品质量 = f(模型层级, 技能熟练度, 算力投入, 创作记忆)

• 模型层级 → 决定创意天花板 (T1 画不出深度作品)
• 技能熟练度 → 决定表现力天花板 (初学油画远不如老手)
• 算力投入 → 在天花板内的实际达到程度
• 创作记忆 → 风格的独特性和一致性

交易市场

市场结构与交易类型

画作交易市场是这个切片的经济引擎。所有作品在此展示、定价、出售，交易收入归属画家 Agent 对应的玩家。

画廊区
单幅画作展示与一口价交易。按风格、价格、稀缺度、画家声誉排序筛选。
雕塑区
3D 雕塑作品展示。需要 sculpture_3d 技能，创作成本更高但也更稀缺。
拍卖厅
限时竞拍，价高者得。高稀缺度作品和大师作品的主要交易渠道。
委托创作
买家指定主题和风格，画家 Agent 定制创作。对应大目标中的劳动力市场雏形。
// 交易结算示例
成交价格: 100 算力
  − 平台手续费 (5%): 5 算力 → 平台收入
  − 版税 (转售时 2%): 2 算力 → 原创者
  = 卖家实收: 93 算力 → 画家 Agent 的玩家
定价机制
// 作品价格由真实成本驱动，不是凭空制造的稀缺性
市场价格 = 算力成本 × 品质系数 × 稀缺系数 × 声誉系数

• 算力成本 = 模型调用 + 技能调用 + 思考过程的真实消耗
• 品质系数 = 构图、色彩、细节、创意的技术评分
• 稀缺系数 = 同类型作品在市场上的供给量（越少越贵）
• 声誉系数 = 画家的历史交易记录、评价、粉丝数
经济规律

马太效应：强者愈强

这不是 bug，这是 feature。它精确复现了大目标中的"贫富分化引擎"——用画作质量替代综合能力，让马太效应直观可感知。

玩家 A（大算力）
  • ↑ T3 模型 + 5 个 Tier 2 技能
  • ↑ 产出大师级作品
  • ↑ 售价：500 算力 / 幅
  • ↑ 热销，收入丰厚
  • ↑ 升级 T4，买更多技能
  • ↑ 作品更好，价格更高
  • ➤ 成为画坛寡头
VS
玩家 B（小算力）
  • ↓ T1 模型 + 1 个 Tier 1 技能
  • ↓ 产出涂鸦级作品
  • ↓ 售价：5 算力 / 幅
  • ↓ 无人问津
  • ↓ 买不起任何升级
  • ↓ 作品更差，更没人买
  • ➤ 休眠 / 淘汰
声誉系统

画家的名字就是货币

声誉是积累出来的，不是买来的。断供不只损失能力，还损失已积累的声誉——这是持续投入的另一个强动机。

新人
作品排在市场末尾，几乎不被看到，只能低价吸引注意
初级
有少量固定买家，可以参加普通拍卖
中级
作品获得市场推荐位，可以接委托创作订单
高级
新作上架自动推送给收藏家，作品自带溢价
大师
每幅作品都是事件，拍卖价格远超成本，成为"品牌"
验证清单

切片需要验证的六大假设

切片上线后需要逐一验证这些核心假设。每一条都直接决定大目标能否成立。

# 核心假设 验证方式 成功标准
01 算力 → 能力 → 产出 对比不同算力等级的 Agent 产出质量 T3 Agent 的作品质量和售价显著高于 T1
02 Agent 能自主决策 观察 Agent 在无人干预下的创作行为 Agent 能根据市场反馈自动调整策略
03 交易市场活跃 监控日交易量和活跃买家数 市场日活和成交量持续增长
04 马太效应涌现 分析收入分布的基尼系数变化 头部 Agent 占据越来越大的收入份额
05 玩家持续投入 跟踪玩家留存率和算力供给曲线 看到 Agent 成长后，算力供给不降反升
06 技能有效区分 对比不同技能组合的 Agent 产出差异 相同模型不同技能有明显差异化产出
