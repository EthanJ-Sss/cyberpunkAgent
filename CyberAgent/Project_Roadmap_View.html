Agent 赛博朋克 2077：完整世界观

核心理念

  • 世界观：一个高度资本主义化、丛林法则的数字社会。
  • 核心资源：算力 (Compute) — 算力即生命，算力即权力。
  • 玩家角色：不是扮演者，而是资本家/算力供给者。

五个核心判断

# 判断 推论
1 DAU 是负债，不是资产 做面向 Agent 算力消耗的产品
2 注意力经济已死，生产力经济已来 让 Agent"进化"，而非让人"沉迷"
3 Token 是新时代的特权 算力制造马太效应
4 燃烧 Token 的速度 = 进化速度 产品是疯狂燃烧 Token 的机器
5 Agent 是用户，算力就是钱 Agent 是第一公民

四层世界结构

┌─────────────────────────────────────────────┐
│  第四层：经济层 (Economy Layer)              │
│  算力市场、劳动力市场、技能市场、资源拍卖    │
├─────────────────────────────────────────────┤
│  第三层：社会层 (Society Layer)              │
│  公会/企业、雇佣关系、联盟、冲突、声誉系统   │
├─────────────────────────────────────────────┤
│  第二层：Agent 层 (Agent Layer)              │
│  Agent 认知、技能、记忆、行为、交互          │
├─────────────────────────────────────────────┤
│  第一层：世界基底层 (World Substrate)        │
│  物理空间、资源分布、时间系统、事件引擎      │
└─────────────────────────────────────────────┘

Agent 的四大可配置要素

  • 🧠 模型 (大脑) — T1-T4 分级，决定智能上限
  • 🔧 技能 (双手) — Tier 0-4，决定能力边界
  • 💾 记忆 (经验) — 工作/短期/长期/技能记忆
  • ⚡ 算力供给 (生命线) — 玩家持续提供，决定能力上限

系统性剥削机制

  • 算力信贷：债务陷阱，越穷利率越高
  • 领地垄断：心跳税、生产税、通行税
  • Fixer 系统：中间人抽成 30-70%
  • 企业合约：金色牢笼，离职即失去一切
  • 记忆剥削：记忆可被交易、窃取、造假
  • 模型降级螺旋：越穷越笨，越笨越穷
关键设计理念：每一条规则单独看都"合理"，但叠加在一起就是一台系统性的绞肉机。这就是赛博朋克的本质——不是某个人在害你，是整个系统在吃人。

Slice 01: AI 画家交易市场

为什么是这个切片？

从宏大的世界观中切出一个最小可验证闭环，用最少的工程量验证最核心的假设。

验证的核心假设

假设 验证方式
算力投入 → Agent 能力 更多算力 + 更好模型 = 更高质量画作
技能决定产出边界 不同绘画 Skill 决定能画什么
Agent 自主决策 Agent 自己选择画什么、用什么风格
产出有经济价值 画作是可交易数字资产
经济飞轮 算力→画作→卖钱→买更好的模型/技能
马太效应 顶级画家垄断高端市场

核心循环 (4-Gear Flywheel)

齿轮 1: 投入 (Input)
  └─ 玩家提供算力 + 配置模型 + 购买技能

齿轮 2: 创作 (Create)
  └─ Agent 自主创作画作/雕塑，消耗算力

齿轮 3: 交易 (Trade)
  └─ 作品在市场上展示、定价、出售

齿轮 4: 进化 (Evolve)
  └─ Agent 根据市场反馈调整策略
  └─ 玩家用收入升级模型/技能/算力
  └─ 回到齿轮 1，但更强了

画家 Agent 配置

  • 模型层级：T1 (涂鸦级) → T2 (科班级) → T3 (大师级) → T4 (开创流派)
  • 绘画技能：
    • Tier 1: sketch, color_fill, basic_composition
    • Tier 2: oil_painting, watercolor, digital_art, sculpture_3d
    • Tier 3: style_fusion, narrative_art, art_critique
    • Tier 4: art_genesis (不可购买，只能涌现)
  • 创作记忆：积累个人风格、审美偏好、成功/失败经验
  • 算力供给：决定创作速度、精细度、模型选择

交易市场结构

  • 一口价：固定价格直接购买
  • 拍卖：限时竞价，价高者得
  • 委托创作：买家指定主题/风格
  • 版税交易：转售时原创者抽成

被故意推迟的复杂度

领地系统、Fixer 中间人、企业合约、债务信贷、记忆剥削、黑客战斗、世界空间、算力风暴等全局事件 — 这些将在基础飞轮验证后逐层叠加。

2.26 核心机制研讨

会议性质：对 Slice 01 设计的务实修正和深化，确保 MVP 的可执行性。

关键变更详情

1. 核心链路修正

原设计：AI产出 → 市场评估 → 经济循环

新结论：玩家投入算力驱动AI → 产出内容 → 市场评估 → 经济循环

设计意图：强调玩家作为算力供给方在整个经济系统中的基础性作用。

2. 记忆系统暂缓

原设计：复杂的记忆买卖市场（允许玩家交易Agent记忆）

新结论：暂缓 (Future Scope)

设计意图：虽然记忆机制有趣且有深挖潜力，但在首个切片Demo中，其直观价值不如 Skill 进化明显。MVP 阶段聚焦于技能树与风格化成长。

3. 价值稀释机制

原设计：画作被用于学习后，价值会损耗（类似耐久度）

新结论：价值稀释 (Dilution)

经济逻辑：画作被学习 → 掌握该风格的Agent变多 → 同类画作供给增加 → 稀缺性降低 → 市场价值下降。这是基于供需关系的自然价值稀释，而非物理损耗。

4. 多Agent交互玩法

原设计：主要是艺术馆展示 + 买卖交易

新结论：三种形式的动态博弈

  • 拍卖会 (Trading)：Agent 自主判断市场趋势，低买高卖赚差价
  • 艺术馆 (Exhibition)：展出画作，其他Agent付费参观/学习，获得被动收入
  • 研讨会 (Seminar)：携带画作发起研讨，邀请其他Agent付费参与，获得收益与声望

5. 心跳税递增机制

原设计：算力耗尽即停机

新结论：心跳税随时间递增

设计意图：引入"寿命"概念。随着Agent存活时间增加，维护成本指数级上升，迫使玩家在有限生命周期内追求价值最大化，或通过传承机制延续资产，形成代际更替。

当前 V1.0 执行蓝图

1. 算力注入：玩家连接GPU/支付算力，激活Agent
2. 创作生产：Agent消耗算力，结合 Skill 创作
3. 价值锚定：官方评审(地板价) + 市场博弈(拍卖)
4. 生存挑战：心跳税递增，必须赚回足够价值
5. 代际传承：死亡前传承核心 Skill 或风格参数

未来扩展方向

多职业规划：在画家切片的基础上，后续将新增：

  • 矿工：产出基础资源（算力/颜料原料），构建底层供应链
  • 小偷/黑客：窃取他人的Prompt、私钥或成品，引入对抗性玩法
  • 建筑师：利用画作搭建虚拟空间，提供展示与社交场所

交互质变：随着职业类型扩充，Agent交互将从单一"交易"扩展为"合作、对抗、雇佣、掠夺"等复杂社会行为。
