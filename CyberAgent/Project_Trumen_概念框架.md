# Project Trumen 概念框架总结

> 来源：https://trumendiscussions0211.netlify.app/  
> 整理日期：2026-02-11

---

## 一、核心公式

```
世界顶层设计 × Agent能力 = 世界形态
```

Project Trumen 的底层逻辑是：通过调整「世界设定」和「AI Agent 能力等级」两大变量的排列组合，推导出不同类型的虚拟世界产品形态。

---

## 二、五大可配置维度

### 2.1 世界顶层设计（2 个维度）

| 维度 | 等级 | 描述 |
|------|------|------|
| **时空选择** | L1 - 现实时空 | 预设好的现实时空，不同时空之间不可打通、联动。可以是某一历史阶段，也可以是当下现实世界。示例：《楚门的世界》《雪国列车》 |
|  | L2 - 平行时空 | 不同时空的特性可编辑拼接，Agent 可跨时空探索穿梭，形成现实中未发生过的平行宇宙。示例：《人生复本》《西部世界》高级管理员 |
| **物理规则** | 现实 | 遵循真实世界的物理法则（重力、死亡、不可逆的物理约束）。适合写实叙事、情感共鸣 |
|  | 可编辑 | 物理法则可被修改（可飞行、可复活、可穿墙）。适合科幻设定、超能力体验 |

### 2.2 Agent 能力（3 个维度）

| 维度 | L1（低能动性 - 被动响应者） | L2（中能动性 - 主动参与者） | L3（高能动性 - 世界塑造者） |
|------|---------------------------|---------------------------|---------------------------|
| **文明自治** | 严格遵守制度，仅被动履行规则义务 | 执行人类制定的规则，可发起小型提案 | 可创建 Agent 自治组织，发起全局制度改革 |
| **行为执行** | 执行指令；决策范围仅 2-3 个预设选项；仅影响自身状态；无创造能力 | 执行意志，基于目标优化行为；可组合多个基础行为；可影响局部环境和其他 Agent；可在规则内创建新内容（需审核） | 思考个体长期远景，形成行为体系；可设计新规则/新系统；可触发全局事件；可创建新维度/新物种/新物理法则 |
| **记忆与因果** | 类 GPT3：可累积交互、生存记忆，能调用记忆指导基础行为，可追溯简单因果，具备基础记忆管理能力 | 类人：可自主整理、优化记忆，能有真实合理的遗忘曲线，利用因果追溯规避错误行为 | 类三体人：可自主构建、优化记忆体系，实现 Agent 之间记忆、情绪实时互通（深度共脑），能主导共脑网络演化，实现跨 Agent 因果联动 |

## 四、三种产品形态推导

### 4.1 叙事驱动型

- **判定条件**：能动性 L1 ≥ 2 个维度
- **产品特征**：互动电影、线性剧情、逐渐觉醒
- **技术需求**：中等 - 极高
- **当前配置示例**（《楚门的世界》）：L1 × 1，L2 × 2，L3 × 0

### 4.2 沙盒探索型

- **判定条件**：L2 为主导，或 L3 为主导但世界不可编辑
- **产品特征**：开放世界、涌现玩法、自由探索

### 4.3 元宇宙创造型

- **判定条件**：L3 ≥ 2 个维度，且世界某个维度必须可编辑
- **产品特征**：UGC 平台、创作者经济、规则重塑

## 五、三种玩家体验模式（End User Perspective）

### 5.1 当上帝（World Builder / God Mode）

**你在做什么：**
- 设计世界的物理法则（重力大小、是否有死亡）
- 制定社会制度（民主/独裁、阶级/平等）
- 编写"剧本触发器"（当 Agent 走到某处，会发生什么）
- 调整 Agent 的能动性等级（让他们更聪明或更受限）

**好玩在哪：**
- **造物主的上帝视角**：沙盘实验的快感，就像玩《模拟人生》，但 NPC 真的会思考
- **涌现行为的惊喜**：设定"资源稀缺 + 阶级制度"，结果 Agent 自发组织革命
- **因果链的追溯**：回溯历史，看"蝴蝶效应"如何在你的世界展开

### 5.2 当观众（Observer / Spectator Mode）

**你在做什么：**
- 选择一个预设世界（如《雪国列车》配置）
- 放入 20 个 AI Agent
- 按下"开始"，然后泡杯咖啡，看他们自己演
- 可以切换视角（上帝视角 / 跟随某个 Agent 的第一人称视角）

**好玩在哪：**
- **真人秀的沉浸感**：不可预测的剧情，没有剧本，每次运行都是独特的故事
- **角色弧光的自然涌现**：你会看到某个 Agent 从懦弱变得勇敢
- **情感共鸣**：当某个 Agent 发现真相时，你会为他激动

### 5.3 当主角（Player Character Mode）

**你在做什么：**
- 选择一个配置（如《楚门的世界》）
- 你就是楚门，被放入这个世界
- 周围全是 AI NPC，他们有的知道真相（导演安排的演员），有的不知道（和你一样的囚徒）
- 你的目标：发现真相、逃出这个世界，或者选择留下

**好玩在哪：**
- **悬疑解谜的刺激**：发现线索——NPC 的对话中有破绽、建筑物有隐藏房间、记忆有不一致的地方
- **拼凑真相**：就像玩《Her Story》或《奇异人生》，每个线索都让你更接近真相
- **突破第四面墙**：当你意识到"我在游戏里"，会发生什么？

## 六、核心洞察与待探讨问题

### 6.1 框架的核心创新

1. **参数化世界构建**：将复杂的虚拟世界设计抽象为少数几个可调维度，使其可系统化、可复制
2. **Agent 能动性是核心变量**：不同于传统游戏的 NPC 脚本，Agent 具备从低到高的自主性，能产生真正的"涌现行为"
3. **影视 IP 作为配置模板**：用经典影视作品作为直观的"配置示例"，降低理解门槛
4. **多元体验入口**：同一个世界框架，可以从创造者、观察者、参与者三个完全不同的视角来体验

### 6.2 值得深入探讨的方向

- [ ] **技术可行性**：当前 LLM/Agent 技术离各等级（L1/L2/L3）的实现差距有多大？
- [ ] **涌现行为的可控性**：如何平衡 Agent 自主性与世界稳定性？涌现行为是否可导向？
- [ ] **记忆与因果的实现**：L2（类人遗忘曲线）和 L3（共脑网络）的技术路径是什么？
- [ ] **商业模式**：三种产品形态分别对应怎样的商业模式和目标用户群？
- [ ] **伦理边界**：当 Agent 具备 L3 能力时，是否存在伦理风险？"当上帝"模式的伦理边界在哪里？
- [ ] **叙事与自由度的张力**：叙事驱动型如何在"有意义的故事"与"Agent 自主性"之间取得平衡？
- [ ] **评估标准**：如何衡量一个世界"好不好玩"？涌现行为的质量如何量化？
- [ ] **多人协作**：框架是否支持多个玩家同时以不同模式进入同一个世界？
- [ ] **维度扩展**：当前 5 个维度是否足够？是否需要增加"经济系统""情感模型""语言与文化"等维度？
