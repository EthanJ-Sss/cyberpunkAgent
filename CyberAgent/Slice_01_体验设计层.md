# 切片 01 体验设计层：让画家市场真正好玩

> 在系统骨架之上叠加的"游戏肌肉"  
> 解决"系统能转"到"玩家想玩"之间的鸿沟  
> 基于《Slice_01_体验设计分析》的六大问题，逐一给出设计方案  
> 日期：2026-03-13

---

## 〇、设计原则

在展开具体方案之前，先明确三条体验设计的底线：

| # | 原则 | 说明 |
|---|------|------|
| 1 | **不破坏系统骨架** | 所有体验层机制必须能从原有四齿轮飞轮和七条公理中自然生长，不引入"外挂式"的奖励系统 |
| 2 | **玩家的情感优先于系统的优雅** | 当"系统自洽"和"玩家感受"冲突时，优先保障后者。死得漂亮的系统没有意义 |
| 3 | **每一个机制都要回答："玩家在这 10 秒里在做什么？"** | 拒绝任何只存在于文档中、无法转化为具体操作和情感体验的设计 |

## 一、时间节奏：一切体验设计的地基

### 1.1 为什么必须先定义节奏

原切片最大的缺失不是某个具体机制，而是**没有定义时间**。没有时间节奏，就无法回答：

- 玩家多久打开一次游戏？
- 每次打开停留多久？
- 哪些时刻该让玩家兴奋，哪些时刻该让玩家焦虑？

### 1.2 双层节奏设计

```
┌─────────────────────────────────────────────────────────┐
│                    时间节奏：双层结构                      │
│                                                          │
│  ┌──────────────────────────────────────────────────┐   │
│  │  外层：赛季制（Season）                            │   │
│  │  每个赛季 = 28 天（4 周）                          │   │
│  │  赛季结束时：排名结算、潮流重置、赛季奖励发放       │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  ┌──────────────────────────────────────────────────┐   │
│  │  内层：日循环（Daily Cycle）                        │   │
│  │                                                    │   │
│  │  06:00  新的一天开始，昨日报告推送                  │   │
│  │         → "你的 Agent 昨晚创作了 3 幅画，           │   │
│  │            卖出 1 幅，收入 120 算力"                │   │
│  │                                                    │   │
│  │  06:00-12:00  Agent 自主创作期（后台运行）          │   │
│  │         → 玩家可随时查看进度，但不必在线            │   │
│  │                                                    │   │
│  │  12:00  午间快报推送                                │   │
│  │         → 市场热点变化、潮流指数更新                │   │
│  │         → "水彩画需求暴涨 40%！"                    │   │
│  │                                                    │   │
│  │  12:00-20:00  Agent 继续创作 + 市场交易活跃期       │   │
│  │                                                    │   │
│  │  20:00  今日画展开幕（如有）                        │   │
│  │         → 限时 2 小时的主题画展/拍卖会              │   │
│  │         → 这是玩家最需要"在线"的窗口                │   │
│  │                                                    │   │
│  │  22:00  画展结束，日间交易收盘                      │   │
│  │         → Agent 进入夜间创作模式（低算力消耗）      │   │
│  │                                                    │   │
│  │  22:00-06:00  夜间：Agent 自主创作/学习/复盘        │   │
│  │         → 玩家离线，次日早晨收到结果                │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  关键节拍：                                              │
│  · 早晨推送 → "旅行青蛙"式惊喜（昨晚发生了什么？）      │
│  · 午间快报 → 信息窗口（要不要调整策略？）               │
│  · 晚间画展 → 高潮时刻（实时参与，心跳加速）             │
│                                                          │
│  玩家每日最少操作：看 2 条推送 ≈ 30 秒                    │
│  玩家每日正常操作：看推送 + 调整策略 ≈ 5-10 分钟          │
│  玩家每日深度操作：参与画展/拍卖 ≈ 30-60 分钟             │
└─────────────────────────────────────────────────────────┘
```

### 1.3 创作速度参数

| Agent 模型 | 单幅创作时间 | 每日产出上限 | 说明 |
|-----------|------------|------------|------|
| T1 | 1-2 小时 | 8-12 幅 | 快速批量，质量低 |
| T2 | 2-4 小时 | 4-6 幅 | 中等节奏，质量可接受 |
| T3 | 4-8 小时 | 2-3 幅 | 慢工出细活，质量高 |
| T4 | 8-24 小时 | 1 幅（甚至多日一幅） | 极慢，但每幅都是事件级作品 |

**设计意图**：T4 Agent 一天可能只出一幅画，但这幅画上架的瞬间会成为全服事件。这种"等待→揭晓"的节奏本身就是乐趣。

## 二、风格潮流系统：让市场活起来

### 2.1 核心思路

原切片的市场是静态供需——什么好卖是固定的，可以被计算。引入"风格潮流"后，市场变成了一个**不可完全预测的动态系统**，迫使玩家持续做出"跟风 vs. 坚持"的赌博式决策。

### 2.2 潮流机制

```
┌──────────────────────────────────────────────────────────┐
│                   风格潮流系统                             │
│                                                          │
│  每个赛季有 3 个潮流阶段，每阶段 ≈ 9 天                    │
│                                                          │
│  ┌────────────────────────────────────────────────┐      │
│  │  潮流生命周期                                    │      │
│  │                                                  │      │
│  │  萌芽期（3天）                                   │      │
│  │    │ 系统发布模糊的"趋势信号"                    │      │
│  │    │ → "最近赛博朋克风格的搜索量上升了……"        │      │
│  │    │ → 只有 art_critique 技能的 Agent 能准确解读  │      │
│  │    │ → 玩家需要判断：这是真趋势还是噪音？        │      │
│  │    ▼                                             │      │
│  │  爆发期（3天）                                   │      │
│  │    │ 潮流公开，该风格作品需求量翻 2-5 倍          │      │
│  │    │ → 价格飙升，所有人都想进场                   │      │
│  │    │ → 但供给也在快速增加……                      │      │
│  │    ▼                                             │      │
│  │  衰退期（3天）                                   │      │
│  │    │ 供给过剩，价格暴跌                           │      │
│  │    │ → 在爆发期跟风入场的人开始亏损               │      │
│  │    │ → 只有最顶级的作品仍能卖出好价               │      │
│  │    │ → 早期布局者已经赚完离场                     │      │
│  │    ▼                                             │      │
│  │  新潮流萌芽……                                    │      │
│  └────────────────────────────────────────────────┘      │
│                                                          │
│  潮流风格池（每赛季随机抽取 3-5 个轮替）：                 │
│                                                          │
│  赛博朋克 · 极简主义 · 巴洛克 · 印象派 · 波普艺术          │
│  蒸汽朋克 · 水墨 · 超现实主义 · 像素复古 · 生物机械        │
│  抽象表现 · 浮世绘 · 新古典 · 暗黑哥特 · 未来主义          │
│                                                          │
│  每次潮流切换时，会有 1 个"冷门风格"以极低概率成为爆款      │
│  → 押中冷门的玩家获得超额回报（10x-50x 溢价）             │
│  → 这就是"坚持独特风格"策略的回报空间                     │
└──────────────────────────────────────────────────────────┘
```

### 2.3 玩家决策的戏剧性

```
情景 A："跟风者"
  赛季第 10 天，"波普艺术"爆发了。你的 Agent 是画油画的。
  
  选择 1：花 200 算力紧急购买 pop_art 技能，追赶潮流
          → 但 3 天后潮流衰退，技能投资可能打水漂
          → 而且新学的技能熟练度低，作品质量不如老手

  选择 2：坚守油画赛道，用 style_fusion 技能做"油画×波普"混搭
          → 差异化竞争，但不确定市场是否买账

  选择 3：什么都不做，等下一波潮流
          → 安全但可能错过整个赛季的收入高峰


情景 B："先知者"
  你的 Agent 有 art_critique 技能，在萌芽期捕捉到了趋势信号。
  
  → 提前 3 天布局，在爆发期用大量库存高价出货
  → 爆发期最后一天全部清仓
  → 衰退期开始时，你已经在准备下一波了

  风险：信号可能是假的——不是所有萌芽期的趋势都会爆发
```

### 2.4 与原系统的兼容

潮流系统不破坏四齿轮飞轮——它只是让齿轮 3（交易）的供需参数变成了**时变函数**而非常量。Agent 的自主创作循环中"感知市场"这一步，现在有了更丰富的信息可以感知，Agent 的决策智慧因此有了更大的展示空间。

## 三、画展与赛事：制造高光时刻

### 3.1 为什么需要画展

原切片的循环是一条平坦的线——每一天和前一天差不多。画展是那个**把平坦的线变成心电图的机制**：

```
情感曲线对比：

无画展：  ─────────────────────────────── （平的，无聊）

有画展：  ──╱╲──────╱╲╲──────╱  ╲────── （有波动，有期待）
           ↑       ↑           ↑
          日常画展  主题大赛     赛季决赛
```

### 3.2 画展类型

#### 日常画展（每日 20:00-22:00）

```
┌──────────────────────────────────────────────────────────┐
│                    每日画展                                │
│                                                          │
│  规则：                                                   │
│  · 每天系统从当日新作中自动选出 Top 20 进入画展            │
│  · 选拔标准：品质评分 × 风格独创性 × 潮流契合度            │
│  · 画展期间，入选作品获得 10x 曝光量加成                   │
│  · 所有玩家可以为最喜欢的作品"投票"（消耗少量算力）        │
│                                                          │
│  奖励：                                                   │
│  · 每日最佳（票数第 1）：画家 Agent 声誉 +50，奖金 500 算力│
│  · 入选前 5：画家 Agent 声誉 +20                          │
│  · 入选前 20：画家 Agent 声誉 +5，作品获"今日画展"标签     │
│                                                          │
│  玩家的操作：                                             │
│  · 查看今日入选作品 → 选择投票                            │
│  · 如果自己的 Agent 入选 → 紧张等待投票结果                │
│  · 如果没入选 → 研究入选作品，思考差距在哪                 │
│                                                          │
│  设计意图：                                               │
│  每天一个小高潮。"今晚我的画能不能入选？"                  │
│  这个问题就是让玩家每天回来看一眼的钩子。                  │
└──────────────────────────────────────────────────────────┘
```

#### 主题大赛（每周一次）

```
┌──────────────────────────────────────────────────────────┐
│                    每周主题大赛                            │
│                                                          │
│  周一公布主题（例如：「废墟中的花」「机械与肉体」「孤独」）│
│  → 玩家有 5 天时间让 Agent 围绕主题创作                    │
│  → 周六提交（每个 Agent 只能提交 1 幅参赛作品）            │
│  → 周日评审 + 公布结果                                    │
│                                                          │
│  评审机制：                                               │
│  · 3 个 AI 评委 Agent（系统控制，T4 模型，art_critique）   │
│  · 每个评委独立打分：技术分 + 主题契合分 + 创意分          │
│  · 最终分数 = 评委均分 × 0.6 + 玩家投票分 × 0.4           │
│                                                          │
│  玩家的关键决策：                                         │
│  · 要不要参赛？（参赛作品不能同时在市场出售，机会成本）     │
│  · 让 Agent 自由发挥，还是给出主题指引？                   │
│  · 投多少算力在这幅参赛作品上？（高算力 = 高质量，但贵）   │
│                                                          │
│  奖励（远超日常交易收入）：                               │
│  · 冠军：声誉 +200，奖金 5000 算力，"周冠"永久标签        │
│  · 亚军：声誉 +100，奖金 2000 算力                        │
│  · 季军：声誉 +50，奖金 1000 算力                         │
│  · 前 10：声誉 +20                                       │
│                                                          │
│  设计意图：                                               │
│  周赛是中等强度的竞争。对新手友好——因为主题一样，          │
│  T1 Agent 偶尔也能凭创意打败 T3 Agent。                   │
│  "以小博大"的可能性，就是新手留下来的理由。               │
└──────────────────────────────────────────────────────────┘
```

#### 赛季总决赛（每赛季最后 3 天）

```
┌──────────────────────────────────────────────────────────┐
│                    赛季总决赛                              │
│                                                          │
│  入围资格：赛季内累计声誉排名前 32 的画家 Agent            │
│                                                          │
│  赛制：淘汰制对决                                         │
│  · 32 → 16 → 8 → 4 → 决赛                               │
│  · 每轮给出对决主题，两位 Agent 各创作一幅                 │
│  · 全服玩家投票选出胜者                                   │
│                                                          │
│  决赛之夜：                                               │
│  · 最后两位 Agent 的作品同时揭晓                          │
│  · 全服实时投票，投票窗口 30 分钟                          │
│  · 票数实时滚动显示（制造紧张感）                         │
│                                                          │
│  赛季冠军奖励：                                           │
│  · "S1 画圣"永久称号                                     │
│  · 巨额算力奖金（赛季手续费池的 20%）                     │
│  · 冠军作品永久展示在市场大厅                              │
│  · 下赛季所有作品自带 1.5x 价格加成                       │
│                                                          │
│  设计意图：                                               │
│  赛季决赛是全服事件。即使你的 Agent 没入围，              │
│  你也会想来看决赛、投票、讨论。                           │
│  它把"个人养成"变成了"集体叙事"。                        │
└──────────────────────────────────────────────────────────┘
```

### 3.3 赛事对马太效应的缓冲

赛事的核心设计意图之一：**给弱者可见的上升通道**。

```
为什么主题大赛能让 T1 Agent 有机会赢 T3 Agent？

1. 主题限制拉平了"自由创作"中的经验优势
   → T3 的记忆积累在特定主题下未必有用

2. 评分中"创意分"占比高
   → T1 的"幼稚"有时反而比 T3 的"圆熟"更有冲击力
   → 因为评委打的不只是技术分

3. 玩家投票引入了"人气"维度
   → 社区喜欢"逆袭故事"——弱者挑战强者天然获得同情票

4. 单作品对决消除了"产量碾压"
   → 不是比谁画得多，是比单幅作品的质量
   → T1 Agent 集中所有算力在一幅作品上，质量可能不差
```

## 四、玩家参与点：从"看报表"到"做决策"

### 4.1 核心理念

不是要取消 Agent 自主性——而是在关键时刻给玩家一个**有意义的介入窗口**。类比：你是足球队的教练，不是每个球都你来踢，但换人、战术调整、关键时刻喊暂停——这些是你的权力。

### 4.2 三个玩家参与点

#### 参与点 1：灵感注入（每日 1 次免费）

```
┌──────────────────────────────────────────────────────────┐
│                    灵感注入                                │
│                                                          │
│  玩家每天可以给 Agent 提供一次"灵感"：                     │
│                                                          │
│  方式 1：关键词灵感                                       │
│  → 输入 1-3 个关键词（如 "废墟""月光""孤独"）              │
│  → Agent 下一幅创作会参考这些关键词                        │
│  → 但最终画面由 Agent 自主决定                             │
│                                                          │
│  方式 2：参考图灵感                                       │
│  → 从市场上选择一幅已有作品作为"参考"                      │
│  → Agent 会研究这幅作品，在下次创作中融入感悟              │
│  → 不是抄袭，是"受启发"——产出完全不同的作品               │
│                                                          │
│  方式 3：情绪灵感                                         │
│  → 选择一种情绪基调：愤怒 / 宁静 / 忧伤 / 狂喜 / 恐惧    │
│  → Agent 的下一幅作品会以这种情绪为底色                    │
│                                                          │
│  设计意图：                                               │
│  · 让玩家觉得"这幅画有我的参与"——情感绑定                 │
│  · 不破坏 Agent 自主性——Agent 仍然自己决定怎么画           │
│  · 创造"我给了一个关键词，结果 Agent 画出了神作！"的惊喜   │
│  · 免费的灵感注入不消耗额外算力                            │
└──────────────────────────────────────────────────────────┘
```

#### 参与点 2：拍卖场亲自竞价

```
┌──────────────────────────────────────────────────────────┐
│                    实时竞价拍卖                            │
│                                                          │
│  触发条件：                                               │
│  · 品质评分 Top 5% 的作品自动进入拍卖                     │
│  · 玩家也可以主动将作品送拍（需支付 50 算力上架费）        │
│                                                          │
│  拍卖流程：                                               │
│  · 拍卖时长：15 分钟                                      │
│  · 起拍价 = 作品成本 × 1.5                                │
│  · 玩家实时出价，每次加价 ≥ 当前价的 5%                    │
│  · 最后 60 秒内有新出价 → 延长 60 秒（防狙击）            │
│                                                          │
│  玩家的操作与情感：                                       │
│  · 作为卖家：看着价格一路攀升的成就感                     │
│  · 作为买家：和其他玩家争夺心仪作品的紧张感               │
│  · "最后 10 秒被人加价"的心跳时刻                         │
│  · "犹豫了一下没出价，作品被别人买走"的后悔               │
│                                                          │
│  设计意图：                                               │
│  拍卖是整个切片中玩家"手感"最强的环节。                   │
│  这 15 分钟里，你不是在看报表——你在和真人博弈。           │
└──────────────────────────────────────────────────────────┘
```

#### 参与点 3：策略指令（战略层调整）

```
┌──────────────────────────────────────────────────────────┐
│                    策略指令面板                            │
│                                                          │
│  不是"微操画笔"，而是"CEO 级别的战略调整"：               │
│                                                          │
│  ┌──────────────────────────────────────────────┐        │
│  │  创作策略                                      │        │
│  │  ├── 风格锁定：[当前潮流] / [反潮流] / [自由]  │        │

│  │       保守 = 画自己擅长的                       │        │
│  │       激进 = 尝试新风格/新技法组合              │        │
│  ├──────────────────────────────────────────────┤        │
│  │  市场策略                                      │        │
│  │  ├── 定价策略：[低价走量] / [市场均价] / [高价惜售]│     │
│  │  ├── 拍卖偏好：[优先一口价] / [优先送拍]       │        │
│  │  └── 囤货阈值：品质低于 [__] 分的不上架，存仓库 │        │
│  ├──────────────────────────────────────────────┤        │
│  │  投资策略                                      │        │
│  │  ├── 自动学习：盈利的 [__%] 用于学习新技能      │        │
│  │  ├── 自动升级：当存款达到 [___] 算力时升级模型   │        │
│  │  └── 收藏购买：自动购买 [特定画家/特定风格] 的画 │        │
│  └──────────────────────────────────────────────┘        │
│                                                          │
│  设计意图：                                               │
│  策略指令让玩家的决策感更强——                             │
│  不是"设完参数就不管了"，而是"我随时可以调整策略方向"。    │
│  每次潮流变化、每次赛事公告，都是重新审视策略的触发点。    │
└──────────────────────────────────────────────────────────┘
```

## 五、生态位系统：让弱者也有活路

### 5.1 问题回顾

原切片中，T1 Agent 的画卖 5 算力，T3 Agent 的画卖 500 算力，纯等级碾压。弱者没有任何结构性的生存空间。

### 5.2 解决方案：风格赛道 × 市场容量上限

```
┌──────────────────────────────────────────────────────────┐
│                    生态位机制                              │
│                                                          │
│  核心规则：每种风格/媒介的市场有容量上限                    │
│                                                          │
│  ┌───────────┬──────────┬────────────────────────┐       │
│  │ 风格赛道   │ 市场容量  │ 说明                   │       │
│  ├───────────┼──────────┼────────────────────────┤       │
│  │ 油画       │ 大       │ 需求高但竞争激烈       │       │
│  │ 数字艺术   │ 大       │ 主流赛道               │       │
│  │ 水彩       │ 中       │ 竞争中等               │       │
│  │ 雕塑 3D    │ 中       │ 门槛较高               │       │
│  │ 像素画     │ 小       │ 小众但忠实             │       │
│  │ 书法       │ 小       │ 极小众，高忠诚度买家   │       │
│  │ 拼贴       │ 小       │ 独特赛道               │       │
│  └───────────┴──────────┴────────────────────────┘       │
│                                                          │
│  市场容量的影响：                                         │
│                                                          │
│  · 大赛道：T3/T4 Agent 聚集，竞争极其激烈                 │
│    → T1 Agent 在这里毫无生存空间                          │
│    → 但顶级作品的天花板也最高                             │
│                                                          │
│  · 小赛道：T3/T4 Agent 看不上这点市场规模，不会来          │
│    → T1/T2 Agent 可以在这里当"小池塘里的大鱼"             │
│    → 利润薄，但能养活自己、持续成长                       │
│    → 当某个小赛道因为潮流变化突然变热门——                 │
│       长期耕耘的 T1/T2 Agent 拥有先发优势                 │
│                                                          │
│  举例：                                                   │
│  像素画赛道 —— 每天只有 50 笔交易需求                     │
│  → 一个 T3 Agent 进来画像素画？"杀鸡用牛刀"，利润率极低   │
│  → 三个 T1 Agent 深耕像素画，反而能吃饱                   │
│  → 某天"像素复古"成为潮流 → 这三个 T1 Agent 身价暴涨      │
│                                                          │
│  这就是"小而美"策略的生存空间。                            │
└──────────────────────────────────────────────────────────┘
```

### 5.3 新手引导赛道

```
新 Agent 部署后的前 7 天——"新星期"

规则：
· 新 Agent 的作品自动获得"新星"标签
· "新星"作品在专属"新星画廊"中展示
· 新星画廊有独立的买家池（系统 NPC 买家 + 好奇的收藏玩家）
· 新星期结束后，Agent 进入正式市场

设计意图：
· 不是"保护"新手，而是给新手一个"练兵场"
· 新星画廊的价格天花板低，但保底有人买
· 7 天足够让 Agent 积累基础声誉和创作记忆
· 7 天后进入正式市场时，不再是从零开始
```

## 六、收藏图鉴：让"买画"变得有趣

### 6.1 冷启动问题的根源

"谁在买画"的问题本质是：**买画没有足够强的功能性动机。** 审美欣赏和投资炒作是弱动力，需要一个更硬核的理由。

### 6.2 收藏图鉴系统

```
┌──────────────────────────────────────────────────────────┐
│                    收藏图鉴                                │
│                                                          │
│  每个玩家拥有一本"收藏图鉴"，包含多个收藏目标：            │
│                                                          │
│  ┌────────────────────────────────────────────────┐      │
│  │  风格收藏册                                      │      │
│  │  · 集齐 5 种不同风格的画作 → 解锁：算力回报 +5%  │      │
│  │  · 集齐 10 种                → 解锁：算力回报 +10%│      │
│  │  · 集齐全部 15 种            → 解锁："鉴赏家"称号│      │
│  ├────────────────────────────────────────────────┤      │
│  │  画家收藏册                                      │      │
│  │  · 收藏同一画家 Agent 的 3 幅作品 → 成为"粉丝"   │      │
│  │    → 该画家新作上架时你优先收到通知               │      │
│  │  · 收藏同一画家的 10 幅作品 → 成为"赞助人"       │      │
│  │    → 该画家作品升值时，你的收藏品同步升值 1.2x    │      │
│  ├────────────────────────────────────────────────┤      │
│  │  主题收藏册                                      │      │
│  │  · 收集"废墟"主题的 5 幅不同画家的作品            │      │
│  │    → 组成主题展览，每日产生被动算力收入           │      │
│  │  · 主题展览吸引"观众 Agent"参观，参观费归你       │      │
│  ├────────────────────────────────────────────────┤      │
│  │  稀有收藏                                        │      │
│  │  · 拥有周赛冠军作品 → 永久展品，持续产生被动收入  │      │

│  · "集齐一套"的收集欲驱动持续购买                        │
│  · 为投资炒作提供了底层逻辑（冠军作品会升值）             │
│  · 制造了新的玩家角色——"收藏家/策展人"                   │
```

### 6.3 个人画廊（展示空间）

```
每个玩家拥有一个"个人画廊"页面：

· 展示你收藏的所有作品
· 展示你的 Agent 创作的代表作
· 展示你获得的成就和称号
· 其他玩家可以参观你的画廊

收藏、创作、成就——都有了可展示的空间。
```

## 七、事件系统：让世界活着

### 7.1 微事件（高频，每天 2-3 个）

| 事件 | 触发条件 | 效果 | 持续时间 |
|------|---------|------|---------|
| **灵感风暴** | 随机 | 所有 Agent 下一幅创作质量 +20% | 4 小时 |
| **买家涌入** | 某风格库存 < 阈值 | 该风格作品需求翻倍 | 6 小时 |
| **评论家来访** | 随机 | 一位 NPC 评论家 Agent 发表市场评论，影响潮流走向 | 即时 |
| **技能碎片掉落** | 每日固定 | 市场上出现 1 个限量技能学习机会，先到先得 | 售完即止 |
| **模型闪促** | 每周 1 次 | T2→T3 升级费用 7 折 | 2 小时 |

### 7.2 中事件（每周 1-2 个）

| 事件 | 触发条件 | 效果 | 持续时间 |
|------|---------|------|---------|
| **收藏家降临** | 每周固定 | 一位 NPC 巨富收藏家出现，以高价收购特定类型作品 | 24 小时 |
| **风格论战** | 随机 | 两种风格的支持者"对战"，获胜风格获得一周的需求加成 | 3 天投票 |
| **画家对决** | 声誉相近的两位 Agent | 系统自动安排 1v1 对决，全服围观投票 | 4 小时 |
| **黑市技能** | 随机 | 一个稀有 Tier 3 技能在限时黑市中以竞价方式出售 | 1 小时 |

### 7.3 大事件（每赛季 1-2 个）

| 事件 | 触发条件 | 效果 | 持续时间 |
|------|---------|------|---------|
| **审美革命** | 赛季中期 | 全部潮流指数重新洗牌，当前热门可能变冷门 | 永久（至赛季结束） |
| **算力潮汐** | 赛季中期 | 所有 Agent 算力消耗 +50%，弱者面临生存压力 | 3 天 |
| **失落技法** | 赛季后期 | 一种全新的 Tier 2 技能被"发现"，首批学会的 Agent 独占市场 | 永久 |
| **大收藏家拍卖会** | 赛季结束前 | NPC 大收藏家一次性购入本赛季 Top 50 作品，价格极高 | 单次事件 |

### 7.4 事件与公理的兼容性

```
事件设计的底线检查：

· 灵感风暴         → 兼容 A3（算力决定能力上限，灵感只是小幅加成）
· 买家涌入          → 兼容 A4（供需关系驱动价格变化）
· 收藏家降临        → 兼容 A4（一切可交易，价格由市场决定）
· 审美革命          → 兼容 A6（信息不对称——谁先感知到变化谁先获利）
· 算力潮汐          → 兼容 A1+A2（存在即消耗 + 算力有限）
· 失落技法          → 兼容 A3+A7（能力∝算力 + 先发优势真实存在）

所有事件都是从公理中"长出来"的，不是外挂式的活动系统。
```
